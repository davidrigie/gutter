## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Cmd+O` | Open file |
| `Cmd+S` | Save |
| `Cmd+P` | Quick open |
| `Cmd+F` | Find |
| `Cmd+H` | Find & replace |
| `Cmd+/` | Toggle source mode |
| `Cmd+\` | Toggle file tree |
| `Cmd+Shift+C` | Toggle comments |
| `Cmd+Shift+M` | New comment |
| `Cmd+Shift+F` | Zen mode |
| `Cmd+Shift+D` | Cycle theme |
| `Cmd+Shift+T` | Focus mode |
| `Cmd+.` | Command palette |

## Try It

1. Select some text and press `Cmd+Shift+M` to add a comment
2. Type `/` to see the slash command menu
3. Press `Cmd+Shift+D` to switch between light, dark, and system themes
4. Click the outline icon in the status bar to see the document outline
