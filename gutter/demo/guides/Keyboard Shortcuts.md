# Keyboard Shortcuts

Complete reference for Gutter keyboard shortcuts.

## File Operations

| Shortcut | Action |
| --- | --- |
| `Cmd+O` | Open file |
| `Cmd+S` | Save file |
| `Cmd+P` | Quick open (fuzzy finder) |

## Editing

| Shortcut | Action |
| --- | --- |
| `Cmd+B` | Bold |
| `Cmd+I` | Italic |
| `Cmd+E` | Inline code |
| `Cmd+K` | Insert link |
| `Cmd+Shift+X` | Strikethrough |
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |

## Navigation

| Shortcut | Action |
| --- | --- |
| `Cmd+F` | Find |
| `Cmd+H` | Find & replace |
| `Cmd+\` | Toggle file tree |
| `Cmd+Shift+C` | Toggle comments panel |

## Comments

| Shortcut | Action |
| --- | --- |
| `Cmd+Shift+M` | New comment on selection |
| `Cmd+Shift+N` | Next comment |

## View

| Shortcut | Action |
| --- | --- |
| `Cmd+/` | Toggle source mode |
| `Cmd+Shift+R` | Reading mode |
| `Cmd+Shift+D` | Cycle theme (light/dark/system) |
| `Cmd+Shift+E` | Export |
| `Cmd+.` | Command palette |

## Slash Commands

Type `/` at the start of a line to see available block types:

- Heading 1, 2, 3
- Bullet List / Numbered List
- Code Block
- Blockquote
- Table
- Math Block
- Mermaid Diagram
- Image
- Horizontal Rule
