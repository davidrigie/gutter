Everything you need, nothing you don't

✎

WYSIWYG Markdown

Write in a clean visual editor that renders formatting as you type. Headings, bold, italic, links, code blocks, tables — all live.

💬

Inline Commenting

Highlight any text and add comments. Stored as portable HTML markers in standard markdown — no proprietary format lock-in.

🔗

Wiki Links

Connect your notes with [[wiki links]]. Autocomplete suggests files as you type. Backlinks show what links to the current page.

📊

Math & Diagrams

Render LaTeX math equations and Mermaid diagrams inline. Write the code, see the result — no context switching.

🔒

Local-First

Your files stay on your machine. No accounts, no cloud sync, no telemetry. Plain markdown files you can version with git.

⚡

Fast & Native

Built with Tauri and Rust. Starts in under a second, uses minimal memory, and feels like a native app on every platform.
