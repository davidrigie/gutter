Markdown with comments.

A markdown editor where you can highlight text and leave comments. Everything stays in plain files on your machine.

Download for Free View on GitHub

What you get

A writing tool with inline commenting built in.

WYSIWYG Markdown

Formatting renders as you type. Headings, bold, links, code blocks, tables, math, diagrams — you see what you get.

Inline Comments

Highlight text and leave a note. Comments are stored as small HTML markers in your markdown, so they work in any renderer.

Wiki Links

Type [[ to link between notes. Autocomplete suggests files as you type, and backlinks show what references the current page.

Math & Diagrams

LaTeX and Mermaid, rendered inline. Write the syntax, see the result. No export step, no separate preview.

Local-First

No accounts, no cloud, no telemetry. Plain markdown files on disk. Works great with git.

Fast & Native

Built on Tauri and Rust. Starts in under a second, light on memory, native on macOS, Windows, and Linux.
